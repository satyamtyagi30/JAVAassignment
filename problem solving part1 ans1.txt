Q1. A class called circle is designed as shown in the following. It contains:

♦ Two private instance variables: radius (of type double) and color (of type String), with a default value of 1.0 and "red", respectively.

♦ Two overloaded constructors.

♦ Two public methods: getRadius() and getArea().

Write a test program that uses Circle class and its features.
ANS:


//file name: Circle.java
package Classtest;

public class Circle
{  
 
final double PI=3.14;

//2 PRIVATE DATA MEMBERS

 private double radius=1.0; 
 private String color="red";  
 
//2 OVERLOADING CONSTRUCTOR

	Circle()
	    {
              this.radius=1.0;
	    }
	
	Circle(double y,String z)
	    {
              this.radius = y;
              this.color  = z; 
	    }

//2 PUBLIC FUNCTIONS

	public void getradius()
                   {
			System.out.println("Radius: "+radius);
                   }
        public void getArea()
		{
			double area= PI * radius * radius;
			System.out.println("Area : "+area);             
 		}

 }  

class  A
{
   public static void main(String[] args)
{
   Circle c1=new Circle();
   Circle c2=new Circle(20,"blue");
   c1.getradius();
   c1.getarea();
   c2.getradius();
   c2.getarea();
}

}
